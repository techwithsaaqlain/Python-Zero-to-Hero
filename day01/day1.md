## 🧠 1️⃣ What is Python?

Python is a **high-level, beginner-friendly programming language**.

It is:

- Easy to read
- Easy to write
- Very powerful
- Widely used in industry

Python is used in:

- Automation
- DevOps
- Web Development
- Data Science
- AI / Machine Learning
- Cloud Engineering

In DevOps, Python helps automate:

- AWS resource creation
- Log processing
- File handling
- CI/CD tasks
- Infrastructure management

## 💻 2️⃣ Installing Python

### Step 1:
Download Python from the official website.

### Step 2:
Install it on your system.

### Step 3 (Important):
Make sure to check:

✅ **Add Python to PATH**

---

### 🔎 Verify Installation

Open Terminal or Command Prompt and run:

```bash
python --version
```

If you see something like:

```
Python 3.x.x
```

Congratulations 🎉 Python is installed correctly.

## 🖨 3️⃣ Your First Python Program

Create a file called:

```bash

```

## 🔎 4️⃣ Understanding `print()`

`print()` is a built-in Python function used to display output.

Example:

```python
print("My name is Saqlain")
print("I am learning Python")
```

Each print statement outputs text on a new line.

---

## 📝 5️⃣ Comments in Python

Comments are notes for humans.  
Python ignores them during execution.

Example:

```python
# This is a comment
print("Learning Python")
```

Use comments to explain your code clearly.
